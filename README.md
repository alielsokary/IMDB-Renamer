# IMDB-Renamer
A script using IMDB to rename a list of episodes in a given directory.

## Third-party modules:
**`requests`, `beautifulsoup4`**
to install use: 
- `pip install requests`
- `pip install beautifulsoup4`
## Usage
Open **Terminal** and `cd` to the script location:

   `./IMDB-Renamer.py`
